## Configuration

### Github Token Setup

Generate a GitHub Personal Access Token to enable repository automation:

1. Visit [GitHub Settings > Developer Settings > Personal Access Tokens](https://github.com/settings/tokens)

2. Click **"Generate new token (classic)"**

3. Select the following required scopes:
   - âœ… `repo` - Full control of private repositories
   - âœ… `workflow` - Update GitHub Action workflows
   - âœ… `user` - Read user profile data
   - âœ… `admin:repo_hook` - Full control of repository hooks

4. **Copy and securely store** your token

5. Save this token in left panel tab "Env" => "GITHUB_TOKEN"

**Step 1: Token Settings Page**

![github_0.png](assets/images/doc/github_0.png)

**Step 2: Select Scopes**

![github_1.png](assets/images/doc/github_1.png)

âš ï¸ **Security Note:** Never commit your token to version control. Always use environment variables or a `.env` file (add to `.gitignore`).

### HeyGen Token Setup

Generate an API key from HeyGen to enable video generation features:

1. Log in to [HeyGen Dashboard](https://app.heygen.com)

2. Navigate to **Account Settings > API Settings**

3. Click **"Generate API Key"**

4. **Copy and securely store** your API key

5. Save this token in left panel tab "Env" => "HEYGEN_TOKEN"

**HeyGen API Settings**

![heygen_0.png](assets/images/doc/heygen_0.png)

âš ï¸ **Security Note:** Keep your API key confidential. Treat it like a password and never share it publicly.
